# Source Code
This folder contains all the React components, styles, and logic for the application.

- `App.jsx`: The main application component.
- `Die.jsx`: Represents a single die component in the Tenzies game, showing its value and handling click events to freeze its state.
- `index.css`: Contains the global styles and module-specific styles for the components.
- `index.jsx`/`main.jsx`: The entry point where the React application is rendered.
